# 🩸 LifeFlo: Student Health & Blood Donation Portal

## 🌟 The Vision
LifeFlo is a specialized module designed for Higher Learning institutions. It integrates directly into student portals to bridge the gap between young donors and national blood banks.

### Key Features
- **Eligibility Checker:** Instant AI-driven quiz to see if a student can donate.
- **Appointment Booking:** Seamless scheduling synced with the academic calendar.
- **Medical Camp Funding:** An alternative path for students to contribute via micro-donations (M-Pesa/Card).

## 🛠️ Tech Stack
- **Backend:** Python / Django / PostgreSQL
- **Frontend:** React (Next.js)
- **API:** Django REST Framework
